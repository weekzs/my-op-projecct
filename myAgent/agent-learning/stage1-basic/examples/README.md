# 📁 第一阶段示例代码说明

这个目录包含了第一阶段的示例和演示代码，帮助你理解Agent的基础概念。

## 📂 文件说明

### demo.py - 主要演示文件
包含3个演示：
1. **基础使用演示** - 对比简单助手和记忆助手的区别
2. **窗口大小对比演示** - 展示不同窗口大小的效果
3. **API连接测试** - 检查环境配置是否正确

### 运行方式

```bash
# 在stage1-basic目录下运行
python examples/demo.py
```

## 🎯 学习建议

1. **先运行demo.py** - 了解整体功能
2. **然后运行simple_chat.py** - 体验无记忆对话
3. **再运行memory_chat.py** - 体验有记忆对话
4. **最后查看源码** - 理解实现原理

## 💡 实验建议

### 实验1：记忆测试
尝试这个对话序列：
```
用户: 我叫张三，是一名医生
用户: 我住在上海
用户: 我的爱好是摄影
用户: 我刚才说了什么？
用户: 我叫什么名字？
用户: 我住在哪里？
```

### 实验2：窗口大小测试
在memory_chat.py中修改window_size：
- window_size=2 (记住很少)
- window_size=5 (默认)
- window_size=10 (记住很多)

观察不同窗口大小的效果。

### 实验3：Token消耗测试
使用/stats命令查看不同窗口大小的Token消耗。

## 🚨 常见问题

### Q: 提示需要配置API密钥
A: 复制.env.example为.env，然后填入你的API密钥

### Q: API调用失败
A: 检查网络连接、API密钥和base_url是否正确

### Q: 回复很长
A: 这是正常的，LLM会详细回答。可以调整max_tokens参数
